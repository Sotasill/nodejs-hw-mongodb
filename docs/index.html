Get list of contacts

Get list of users with this endpoint

Authorizations:
bearerAuth
query Parameters
page
integer
Example: page=1
perPage
integer
Example: perPage=10
sortBy
string
Example: sortBy=name

All contact's fields can be mentioned. Default value - _id

sortOrder
string
Enum: "asc" "desc"
Example: sortOrder=asc
